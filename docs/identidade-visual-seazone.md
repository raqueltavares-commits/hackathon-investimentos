# Identidade visual Seazone (BrandBook)

Extraído do `BrandBook Seazone.pdf`. Usar no dashboard.

## Tipografia
**Helvética** (única fonte da marca), variando pesos: Thin · Light · Medium · Bold.
Web: usar `"Helvetica Neue", Helvetica, Arial, sans-serif`.

## Paleta principal (usar no dashboard)
| Uso | Hex |
|-----|-----|
| Azul primário (destaque, botões, links) | `#0054FC` |
| Navy escuro (texto, fundos escuros) | `#000C3C` |
| Coral / casinha (accent, só pontual) | `#F06054` |
| Azul claro (estados, hovers) | `#6090FC` |
| Azul muito claro (fundo complementar) | `#E8EFFF` |
| Branco / off-white (fundo) | `#FFFFFF` / `#F7F9FC` |

## Logo
- "seazone" minúsculo; o "a" é a **casinha** (house + círculo). Accent coral.
- Versões: sobre branco, sobre azul `#0054FC`, negativo sobre navy `#000C3C`.

## Cores de apoio — NÃO usar no dashboard
Verdes (`#249555`, `#2BBD68`, `#77DBA4`) e amarelos/laranjas (`#D88800`, `#FAA200`,
`#FCC74D`) são **exclusivos de CTAs/marketing** e precisam de aprovação do Marketing.
Ferramenta interna fica na paleta principal (azul/navy/coral/branco).

## Tom
"Seu lugar fora de casa". Limpo, claro, hierarquia por peso de fonte, muito espaço em branco.
